The Odinbook

Odinbook is an attempt to clone the core functionality of facebook. This lets you connect with your friends. It requires you to create an account first. Once signed in you may share a post which will be visible to your friends, also you may comment, like or save any post. It also lets you find other users on odinbook, send friend request to them and accept/reject friend request of others. We can go to other users' profile see their public details and posts and edit our own profile. This project follows JAMstack architecture where one part of this project serves APIs while other one which is the frontend consumes those APIs and show data to user in a user-friendly and interactive manner.

  • CSS
  • Javascript
  • React
  • Node
  • Express
  • Passport
  • Mongodb
